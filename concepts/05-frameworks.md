# The Framework Toolbox

## How to treat a framework

A framework is a structural lens for understanding and solving a problem — **a tool for looking at the whole MECE-ly and organizing it.**

Any framework (or one you invent) is fine, as long as it structures the situation MECE-ly for *this* problem.

Two warnings:

- **Applying a framework is not the analysis.** You still have to *connect* the organized pieces into an insight.
- **No box-filling.** Don't cram facts into cells to look complete. Drive the elements toward a converging conclusion.

## Six MECE frameworks

| Framework | Use | Parts |
|---|---|---|
| **3C / 4C** | Market/business structure from the players' angle | Customer / Competitor / Company (+Channel) |
| **4P (+7P)** | Break down product/marketing strategy | Product / Price / Place / Promotion (+People, Process, Physical evidence) |
| **SWOT** | The firm's whole situation, internal vs. external | Internal (S/W) × External (O/T), positive/negative |
| **AARRR** | Digital/growth strategy, the customer-journey funnel | Acquisition / Activation / Retention / Revenue / Referral |
| **PEST / PESTLE** | Scan the macro external environment | Political / Economic / Social / Technological (+Legal / Environmental) |
| **Value Chain** | Decompose value creation, find the intervention point | Sourcing → production → distribution → sales → after-sales |

## Which lens for which situation

- Don't know who's even in the market → **3C/4C**
- Want to see marketing/product strategy with no gaps → **4P**
- Want to sort out our own company's position → **SWOT**
- Hunting a growth bottleneck in an app/service → **AARRR**
- Need to see the big currents — regulation, tech shifts → **PEST**
- "Where can I intervene / invest?" → **Value Chain**
- Want the stage-by-stage friction before a customer buys → **purchase-journey / funnel** (arrive / browse / cart / checkout / return)

## Industry analysis

A useful drill: take an unfamiliar industry and **connect the four PEST elements to each other → derive the KSFs** (Key Success Factors = the insight). Not box-filling — connection converging on the KSFs.
